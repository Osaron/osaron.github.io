Content API

Complete reference for managing and retrieving content including blogs, documentation, user manuals, and white papers. All endpoints support filtering, pagination, and full CRUD operations.

List Content

Retrieve content with advanced filtering

Create Content

Content API Base

https://api.oscarrondon.com/v1/content
GET /v1/content Retrieve all content items

Query Parameters

Parameter Type Required Description
type string No Filter by content type: blog, manual, whitepaper, strategy
status string No Filter by status: draft, published, archived
tags string No Comma-separated list of tags
limit integer No Items to return (default: 10, max: 100)
offset integer No Items to skip for pagination

Example Request

curl -H "Authorization: Bearer YOUR_API_KEY" \
     "https://api.oscarrondon.com/v1/content?type=blog&limit=20"
const response = await fetch('https://api.oscarrondon.com/v1/content?type=blog&limit=20', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});
const data = await response.json();

POST /v1/content Create new content

Request Body

Field Type Required Description
title string Yes Content title (max 200 chars)
type string Yes Content type: blog, manual, etc.
content string Yes Main content body (Markdown supported)
tags array No Array of tag strings
status string No draft (default) or published

Example Request

curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Advanced API Documentation",
    "type": "blog",
    "content": "# Advanced API Documentation\n\nThis covers...",
    "tags": ["api", "documentation"],
    "status": "published"
  }' \
  https://api.oscarrondon.com/v1/content
const newContent = {
  title: "Advanced API Documentation",
  type: "blog",
  content: "# Advanced API Documentation\n\nThis covers...",
  tags: ["api", "documentation"],
  status: "published"
};

const response = await fetch('https://api.oscarrondon.com/v1/content', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(newContent)
});
