GET random.php

Returns a single random meal object. No parameters required. Each call returns a different meal.

JavaScript
const res  = await fetch('https://www.themealdb.com/api/json/v1/1/random.php');
const data = await res.json();
const meal = data.meals[0];

Returns all meal categories with their IDs, thumbnails, and descriptions. Useful for populating a category selector.
