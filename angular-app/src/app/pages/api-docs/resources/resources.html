Basic Examples

Fetching and Displaying Content

import React, { useState, useEffect } from 'react';
function ContentList() {
  const [content, setContent] = useState([]);
  useEffect(() => {
    client.content.list({ type: 'blog', status: 'published', limit: 10 })
      .then(r => setContent(r.data));
  }, []);
  return content.map(item => <article key={item.id}><h2>{item.title}</h2></article>);
}
export default {
  data() { return { content: [] }; },
  async mounted() {
    const r = await client.content.list({ type: 'blog', status: 'published' });
    this.content = r.data;
  }
};
const response = await fetch('/v1/content?type=blog&status=published', {
  headers: { 'Authorization': `Bearer ${apiKey}` }
});
const { data } = await response.json();
data.forEach(item => console.log(item.title));

Advanced Examples

Real-time Content Updates with WebSockets

class RealTimeManager {
  constructor(apiKey) {
    this.ws = new WebSocket(`wss://api.oscarrondon.com/v1/ws?token=${apiKey}`);
    this.ws.onmessage = (e) => this.handleMessage(JSON.parse(e.data));
  }
  handleMessage(msg) {
    if (msg.type === 'content.updated') this.updateUI(msg.data);
  }
}

Frequently Asked Questions

Common questions and answers about using our API.

General Questions

What are the rate limits?

Rate limits depend on your plan: Free 100 req/hr, Basic 1 000 req/hr, Pro 10 000 req/hr, Enterprise custom limits. Rate limit headers are included in every response.

How do I handle authentication securely?

Never expose API keys in client-side code. Use environment variables, rotate keys every 90 days, and implement IP whitelisting. For frontend apps, create a backend proxy that handles API authentication.

What data formats are supported?

The API exclusively uses JSON (UTF-8, RFC 7159). All endpoints accept and return application/json.

How do I paginate through large result sets?

# First page
curl "https://api.oscarrondon.com/v1/content?limit=10&offset=0"
# Second page
curl "https://api.oscarrondon.com/v1/content?limit=10&offset=10"

The response includes pagination.has_more to detect if more pages exist.

Technical Questions

Can I use webhooks for real-time notifications?

Yes! Webhooks are available for Pro and Enterprise plans. Configure them in your dashboard under "API Settings".

Is there API versioning?

Yes, URL-based versioning (/v1/). We maintain backwards compatibility for at least 12 months and announce breaking changes in advance.

Billing & Plans

How is API usage calculated?

Each successful (2xx) request counts toward your quota. Failed requests (4xx/5xx) don't count. Usage resets monthly on your billing cycle date.

Can I upgrade or downgrade my plan?

Yes. Upgrades take effect immediately; downgrades take effect at the next billing cycle. No penalties for plan changes.
